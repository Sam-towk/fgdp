## PERGUNTAS SOBRE HTML

1. O que significa HTML?

resposta

Hyper Text Markup linguage

2. O que são TAGS em HTML?

resposta

Componentes visuais para construir a interface da pagina

3. Como define uma TAG no HTML?

resposta

uma tag é defina atravez dos simbolos "maior que" e "menor que", dentro desses simbolos é colocado uma letra ou palavra que vai definir a função da tag exemplo

< p >

4. O que são atributos em HTML?

resposta

Os atributos são parametros dentro das tags que possuem uma função similar a condições de entrada e saida da mesma.

Exemplo: Voltada a programação em ladder ela é similar a o parametro de operação de um componente

5. Qual a estrutura básica em HTML?

resposta

Uma estrutura basica de html é composta por html, head, body e tittle.

6. Explique as TAGS abaixo:

a. HTML

resposta

É a tag que define o inicio e o fim de um documento HTML

b. HEAD

resposta

É a tag que define as configurações iniciais para configurar o ambiente que vai ser trabalhado

c. META

resposta

É a tag que define as configurações de metadados do documento HTML

d. TITLE

resposta

É a tag que define o titulo da pagina

e. BODY

resposta

É a tag que define o corpo do documento, aonde vai ser contruido todo o miolo da pagina

f. H1

resposta

É a tag que define o titulo principal da pagina, so devendo usa-lo apenas uma vez

g. OL

resposta

É uma tag para definição de lista sendo ela a enumerada/p>

h. LI

resposta

É a tag usada dentro da tag OL para fazer a lista, recebendo a configuração da tag mãe(ol)

## PERGUNTAS SOBRE CSS

1. O que significa CSS?

resposta

Cascading Style Sheets

2. O que é SELETOR em CSS?

resposta

É o que vai definir qual elemento vai ser custumizado no documento de css

3. Como define quais são os SELETORES no CSS?

resposta

Dependendo da tag adereçada o seletor pode ser definida pelo nome, dois pontos, hashtags, ponto, tipo e dois, dois pontos

4. Qual a estrutura básica em CSS?

resposta

Uma estrutura padrão de css é seguida da tag ou identidade do objeto, abertura de chave, customização da tag separada por ponto e virgula e fechamento da chave

5. Qual a hierarquia de prioridade nos seletores em CSS?

resposta

Estilo in line no propio documento html, seletores id (que so podem ser adereçados uma vez), atributos e classes e por ultimo os elementos

6. Explique para que servem as PROPRIEDADES abaixo:

a. width, height

resposta

Define a largura e altura do elemento adereçado

b. font-size

resposta

Define o tamanho da fonte

c. text-decoration

resposta

Define o tipo de decoração que vai ser aplicada ao texto, como: sublinhado, traçado riscado entre outros

d. text-transform

resposta

Define o tipo de transformação que vai ser aplicada ao texto, como: caixa alta, caixa baixa, capslock entre outros

e. color

resposta

Define a cor do elemento

f. border

resposta

Define a borda do elemento, podendo ser configurada a espessura, cor e estilo

g. margin

resposta

Define o espaço de margem entre o elemento e o restante do documento(global)

h. padding

resposta

Define o espaço entre o elemento e a borda do mesmo(interno)
