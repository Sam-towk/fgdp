## PERGUNTAS SOBRE HTML

1. O que significa HTML?

resposta

Hyper Text Markup linguage

2. O que são TAGS em HTML?

resposta

Componentes visuais para construir a interface da pagina

3. Como define uma TAG no HTML?

resposta

uma tag é defina atravez dos simbolos "maior que" e "menor que", dentro desses simbolos é colocado uma letra ou palavra que vai definir a função da tag exemplo

< p >

4. O que são atributos em HTML?

resposta

Os atributos são parametros dentro das tags que possuem uma função similar a condições de entrada e saida da mesma.

Exemplo: Voltada a programação em ladder ela é similar a o parametro de operação de um componente

5. Qual a estrutura básica em HTML?

resposta

Uma estrutura basica de html é composta por html, head, body e tittle.

6. Explique as TAGS abaixo:

a. HTML

resposta

É a tag que define o inicio e o fim de um documento HTML

b. HEAD

resposta

É a tag que define as configurações iniciais para configurar o ambiente que vai ser trabalhado

c. META

resposta

É a tag que define as configurações de metadados do documento HTML

d. TITLE

resposta

É a tag que define o titulo da pagina

e. BODY

resposta

É a tag que define o corpo do documento, aonde vai ser contruido todo o miolo da pagina

f. H1

resposta

É a tag que define o titulo principal da pagina, so devendo usa-lo apenas uma vez

g. OL

resposta

É uma tag para definição de lista sendo ela a enumerada/p>

h. LI

resposta

É a tag usada dentro da tag OL para fazer a lista, recebendo a configuração da tag mãe(ol)
